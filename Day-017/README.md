## Problems Solved

### 1. Palindrome Linked List (LeetCode #234)

Check whether a linked list reads the same forward and backward.

### Approach

- Store linked list values in an array.
- Compare the array with its reverse.

### Complexity

- Time Complexity: O(n)
- Space Complexity: O(n)

---

### 2. Detect Cycle in a Linked List

Check whether a linked list contains a cycle.

### Approach

- Use slow and fast pointers.
- Slow moves one step.
- Fast moves two steps.
- If both meet, cycle exists.

### Complexity

- Time Complexity: O(n)
- Space Complexity: O(1)
