## Problems Solved

### 1. Merge Two Sorted Lists (LeetCode #21)

### Problem Statement

Merge two sorted linked lists into one sorted linked list.

### Approach

- Create a dummy node.
- Compare both linked lists.
- Attach the smaller node.
- Continue until one list finishes.
- Attach the remaining nodes.

### Complexity

- Time Complexity: O(n + m)
- Space Complexity: O(1)

---

### 2. Reverse a Singly Linked List

### Problem Statement

Reverse the given singly linked list.

### Approach

- Use three pointers:
  - previous
  - current
  - next_node
- Reverse each link one by one.

### Complexity

- Time Complexity: O(n)
- Space Complexity: O(1)
