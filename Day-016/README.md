## Problems Solved

### 1. Swap Nodes in Pairs (LeetCode #24)

### Problem Statement

Given the head of a linked list, swap every two adjacent nodes and return the modified linked list.

### Approach

- Create a dummy node.
- Process the linked list in pairs.
- Swap pointers without changing node values.
- Continue until fewer than two nodes remain.

### Complexity

- Time Complexity: O(n)
- Space Complexity: O(1)

---

### 2. Delete Last Node of a Doubly Linked List

### Problem Statement

Delete the last node of a doubly linked list and return the updated head.

### Approach

- Traverse to the last node.
- Disconnect it from the previous node.
- Return the head.

### Complexity

- Time Complexity: O(n)
- Space Complexity: O(1)
